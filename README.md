# 前端面试考察系统

基于 Vue 3 + TypeScript + Vite + Element Plus 的现场面试考察项目。

## 使用方式

```bash
npm install
npm run dev
```

## 项目说明

左侧菜单为题目目录（13 个分类），右侧为题目页面。题目按面试高频程度排序。

- **代码题**：候选人在 `.vue` 文件中找到 `TODO` 标记编写代码，部分题目支持在线编辑运行
- **问答题**：面试官与候选人口头讨论，折叠面板内有参考答案
